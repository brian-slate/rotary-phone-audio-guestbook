## Documentation Structure

### Hardware Setup
Start with [materials.md](materials.md) to understand what you'll need, then follow [hardware.md](hardware.md) for step-by-step wiring instructions.

### Software Installation
After hardware is assembled, see the main [README](../README.md) for installation options (pre-built image or manual setup).

### Configuration
Once installed, refer to [configuration.md](configuration.md) for detailed explanations of all settings in `config.yaml`.

### AI Features
If you want to enable automatic transcription and metadata extraction, see [ai-integration.md](ai-integration.md) for setup instructions and API details.

### Development
For contributors or those modifying the code, [development.md](development.md) covers the development workflow, dependency management, and deployment procedures. [deployment.md](deployment.md) explains the deployment scripts in detail.
